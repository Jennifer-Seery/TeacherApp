Frequently Asked Questions
  1. How to login : Click on the webpage button on the main screen of this page.

  2. Where are the logins: Logins for the database are found in the logins section of the homepage.

  3. Can this application be used offline: The database section of this application cannot be used offline the rest is free to use offline.

  4. Is there a limit to the amount of students you can add: No.

  5. How to delete a student: The action drop down menu allows a user to delete a student from the database.

  6. Can a deleted person be retrieved : No

  7. How to change a students password: Click on the user you wish to alter and change the password contained in the password field.
